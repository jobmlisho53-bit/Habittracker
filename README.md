# Disciplined Trader

A personal habit tracker app for maintaining strict trading discipline and daily routines.

## Features

- ✅ Track daily routine (Prayer, Exercise, Book reading, etc.)
- ✅ Monitor trading sessions (3 per day: 17:00-17:45, 17:45-18:30, 18:30-20:00)
- ✅ Real-time progress tracking
- ✅ Local SQLite database (no internet required)
- ✅ Automatic APK building via GitHub Actions

## Daily Schedule

- 04:00-04:30: Prayer
- 04:30-05:30: Exercise  
- 05:30-06:00: Book Reading
- 06:00-06:30: Bath & Clothes
- 06:30-07:00: Breakfast
- 07:00-17:00: Classes
- 17:00-17:45: Trading Session 1
- 17:45-18:30: Trading Session 2  
- 18:30-20:00: Trading Session 3

## Tech Stack

- **Frontend**: Flutter (Dart)
- **Database**: SQLite (local)
- **Build**: GitHub Actions (automatic APK)

## Setup

1. Clone repository
2. Run `flutter pub get`
3. Run `flutter run`

## GitHub Actions

The workflow automatically builds APK on every push to main branch.
